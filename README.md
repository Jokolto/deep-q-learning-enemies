# Implementierung von Deep-Q-Learning für adaptive Gegner in einem Spiel.
## Das Ziel
Dieses Projekt wurde im Rahmen eines Projekts am Gymnasium durchgeführt. Es ist der Nachfolger des Maturaprojekts, das sich in einem anderen Repository befindet. Das Ziel besteht darin, das adaptive Verhalten der Gegner mithilfe des Deep-Q-Learning-Algorithmus zu optimieren. Es wurde auch überlegt, an dem früheren Repository weiterzuarbeiten oder es zu forken. Letztlich wurde jedoch entschieden, ein neues Repository zu erstellen, um das Projekt für die Bewertung übersichtlicher zu gestalten.
